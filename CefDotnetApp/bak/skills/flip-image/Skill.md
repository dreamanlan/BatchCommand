# flip-image Skill

Flip an image horizontally or vertically using Python PIL.

## Usage

```
call_skill('flip-image', image_path, flip_direction)
```

## Parameters

- `image_path`: Path to the image file
- `flip_direction`: 0 = horizontal flip, 1 = vertical flip

## Requirements

- Python 3.x
- Pillow library (`pip install Pillow`)
